***Uninstall Procedure for Cores/HMODS***

1.  Make hakchi/transfer directory

2.  Copy uninstall file into it

3.  Boot system

4.  It will auto-shutdown when done

5.  Make hakchi/transfer directory

6.  Copy Reinstall hmods, as well as whichever other hmods
you would like installed!  Boot for double splash screen

7.  Enjoy:)
