

REminiscence is a Port of Gregory Montoir's Flashback emulator, running as a libretro core

1.  Install Core

2.  You will need the ORIGINAL PC Version of Flashback from 1993
(Not included due to legal reasons)

3.  Add .exe as a Game, via Hakchi2 CE, and command line is /bin/reminiscence or /bin/flashback

4.  Copy the rest of the files into the same directory that .exe installs to'
(make sure .exe is NOT compressed by Hakchi2 CE!, or it will not run)

5.  Copy REminiscence data files to data folder

6.  Enjoy this fantastic Port of the originally amazing Delphine PC Classic!
